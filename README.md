# simple-music-player

简陋的音乐播放器demo<br/>
已实现功能：<br/>
<ul>
    <li>展示歌曲列表</li>
    <li>点击歌曲切换到相应歌曲播放</li>
    <li>播放暂停按钮</li>
    <li>上一曲下一曲按钮</li>
    <li>可拖动进度条</li>
    <li>音量大小控制条</li>
    <li>显示歌词按钮</li>
    <li>显示歌词并高亮当前播放到的歌词</li>
    <li>点击加载更多歌曲</li>
</ul>
